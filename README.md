## Tools used

- [Ai21](https://www.ai21.com/)
### Running Locally

- git clone the repo
- ``pip install -r requirements.txt``
- create Ai21 account
- create ``.env`` file inside project directory and write
        ``AI21LAB_KEY=<your_ai21_key>``
- run ``python main.py``
